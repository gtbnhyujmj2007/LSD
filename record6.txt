✅ 原型功能概要 v0.1
🎯 目標
找出 低價位但具備複質押機會的代幣，依照你設定的策略 ETH > SOL > Others 做排序與評估，發出🟢黃🔴訊號。

📦 功能模組設計
模組	功能說明
1. 幣種清單建構	自動或手動載入候選幣種（預設10種含 ETH/SOL）
2. 價格與 LSD 檢查	用 Coingecko API + DefiLlama API 取得價格、是否有 LSD、是否可再質押
3. 價格比較分析	目前幣價 vs. 30日平均、vs ETH 價格比、TVL 增減趨勢
4. 訊號燈評分	每個幣依據分析結果打分並標示綠燈/黃燈/紅燈
5. 輸出建議	顯示可考慮進場的幣種與推薦理由

🔌 使用 API（免註冊）
🧮 Coingecko API：幣價、市值、價格趨勢

📊 DefiLlama API：TVL、LSD 支援協議、收益率

🚨 可選擇：串接 Chainlink 預言機（如果你未來想連實盤）
